# Гид по Логике: «Мозги» вашей стратегии

Логические блоки в DepthSight не анализируют рынок сами по себе. Они управляют тем, как другие условия объединяются между собой. Ниже разбор всех 5 блоков в формате «Что-Зачем-Пример».

---

## 1. AND Block (Логическое И)
*   **Что это**: Блок-контейнер, который требует выполнения **всех** вложенных условий одновременно.
*   **Зачем**: Чтобы отсеять случайные сигналы и входить только тогда, когда есть несколько подтверждений.
*   **Пример**: Положите внутрь `RSI < 30` и `Плотность в стакане`. Бот купит только тогда, когда цена и дешевая, и защищена крупным ордером.

## 2. OR Block (Логическое ИЛИ)
*   **Что это**: Блок-контейнер, которому достаточно выполнения **хотя бы одного** из вложенных условий.
*   **Зачем**: Для создания гибких стратегий, которые могут заходить по разным сценариям.
*   **Пример**: Положите внутрь `Пробой Хая вчера` и `Сигнал от TradingView`. Бот зайдет, если случится либо одно, либо другое.

## 3. Senior TF Confluence (Старший таймфрейм)
*   **Что это**: Специальный контейнер, который проверяет условия на таймфрейме выше вашего рабочего (например, на 1ч вместо 1м).
*   **Зачем**: Чтобы никогда не торговать против «большого» тренда. Если на часе все падает, покупать на минутах — плохая идея.
*   **Пример**: Вложите индикатор тренда в этот блок и выберите "1 Hour". Теперь на 1-минутке бот будет покупать только если на 1-часовике тоже тренд вверх.

## 4. Value Comparison (Сравнение величин)
*   **Что это**: Универсальный инструмент для сравнения двух любых чисел в системе (цены, индикаторы, результаты блоков).
*   **Зачем**: Позволяет создавать свои уникальные правила, которых нет в стандартных блоках.
*   **Пример**: "Если текущая цена выше, чем цена закрытия 5 свечей назад".

## 5. Price vs Level (Цена против Уровня)
*   **Что это**: Специфический блок для сравнения текущей цены с конкретным уровнем, найденным другим блоком.
*   **Зачем**: Позволяет строить логику «пробоя» или «отбоя» от динамических уровней.
*   **Пример**: "Если цена выше уровня из блока Significant Level более чем на 0.2%".
